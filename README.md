## 🎮 Commandes Disponibles

### 💍 Mariage (Marriage)
```
.marry @user          - Demande en mariage
.acceptmarry          - Accepter la demande
.declinemarry         - Refuser la demande
.couple               - Voir le profil du couple
.divorce              - Divorcer
.anniversary          - Voir l'anniversaire
```

### ❤️ Amour (Love)
```
.love @user           - Envoyer de l'amour
.kiss @user           - Embrasser
.hug @user            - Étreinte
.date @user           - Rendez-vous
.lovestats            - Statistiques d'amour
.compatibility @user  - Compatibilité
```

### 👶 Famille (Family)
```
.brother [@user]      - Profil du frère
.sister [@user]       - Profil de la sœur
.family               - Profil familial
.familytree           - Arbre généalogique
.adopt @user          - Adopter quelqu'un
.children             - Voir les enfants
.parent               - Voir les parents
.siblings             - Voir les frères/sœurs
```

### 🎮 Jeux (Games)
```
.couplequiz           - Quiz de couple
.couplebattle @user   - Bataille de couple
.familyquest          - Quête familiale
.couplehunt           - Chasse de couple
.couplework           - Travail de couple
```

### 🏆 Classements (Leaderboards)
```
.topcouples           - Top couples
.topfamilies          - Top familles
.richest              - Les plus riches
.toplove              - Plus aimés
.toplevels            - Niveaux les plus élevés
```

### 📱 Informations (Info)
```
.owner                - Contacter le propriétaire
.contact              - Support
.help                 - Voir toutes les commandes
```

1. Tapez `.help` pour voir toutes les commandes
2. Tapez `.marry @user` pour proposer mariage
3. Tapez `.family` pour voir votre profil familial
4. Tapez `.topcouples` pour voir les classements
5. Contactez `.owner` pour toute aide
